BugTracker

This is a web application, a simple intuitive version of the bug tracker for single use without unnecessary (sometimes useless) things. Register and use it, without long attempts to figure out how, what and why works. In addition, here you will find Drag'n'drop of task cards on the project board, and convenient customization of the application for yourself: color mode (Light/Dark/Auto), language (English or Russian) and time zone.

Used techs: Python, Django, MySQL, Redis, HTML, CSS, Bootsrap and JS.

BugTracker REST API

This project is a connecting element of BugTracker and BugTracker TG Bot, because it allows me to create a telegram bot based on data from BugTracker.

Used techs: Python, Django REST Framework, Django, Redis and MySQL.

BugTracker TG Bot

This is an asynchronous telegram bot that has all the basic functionality of BugTracker, but in the form of a bot: project management and task management for these projects. It is also possible to change the language and time zone.

Yes, I know that this is an asynchronous bot that makes requests to a synchronous API...

Used techs: Python, Aiogram, PostgreSQL, SQLAlchemy, Redis, BugTracker API and requests.
